# Avalab.ai 一括ダウンロード Chrome拡張機能

Avalab.ai で生成した画像とプロンプトを一括でZIPダウンロードできるChrome拡張機能です。

## インストール方法

### 1. ファイルの準備

このフォルダ (`avalab-save`) をPC上の任意の場所に配置してください。

### 2. Chromeに読み込む

1. Chromeで `chrome://extensions/` を開く
2. 右上の **「デベロッパーモード」をON** にする
3. **「パッケージ化されていない拡張機能を読み込む」** をクリック
4. `avalab-save` フォルダを選択
5. 拡張機能一覧に「Avalab.ai 一括ダウンロード」が追加されます

> **注意:** フォルダを移動・削除すると拡張が動作しなくなります。固定の場所に置いてください。

## 使い方

### Step 1: avalab.ai にログイン

1. Chromeで https://avalab.ai/ にアクセスしてログインする
2. https://avalab.ai/generate ページを開く（生成画像一覧が表示されるページ）

### Step 2: 拡張機能を起動

1. Chromeツールバーの拡張機能アイコン（紫色）をクリック
2. **新しいタブ** で操作画面が開きます

### Step 3: トークンを取得

- 画面上部に **「状態: ログイン済み（トークン取得済）」** と表示されれば準備完了です
- **「トークン未取得」** と表示された場合：
  1. avalab.ai のタブに戻り、**F5でページをリロード** する
  2. 拡張機能のタブに戻り、**「再取得」ボタン** をクリック
  3. 「ログイン済み（トークン取得済）」に変わればOK

> **ポイント:** avalab.ai のページを表示・操作すると、拡張機能が認証トークンを自動的にキャプチャします。

### Step 4: 画像一覧を取得

1. **「取得件数上限」** を選択（50件 / 100件 / 500件 / 全件）
2. **「画像一覧を取得」** ボタンをクリック
3. サムネイル付きの画像一覧が表示されます

### Step 5: ダウンロードする画像を選択

- デフォルトで全画像にチェックが入っています
- 個別にチェックを外して選択/解除できます
- **「全選択」「全解除」** ボタンで一括操作も可能です

### Step 6: ZIPダウンロード

1. **「選択した画像をZIPダウンロード」** ボタンをクリック
2. 進捗バーが表示されます（50件ごとにURLを再取得しながらダウンロード）
3. 完了するとZIPファイルが自動的にダウンロードされます
4. **成功した画像はチェックが外れ、失敗した画像だけチェックが残ります**

### 失敗した画像のリトライ

ダウンロードに失敗した画像がある場合：

- **方法1:** ダウンロード完了後、失敗した画像だけチェックが残っているので、そのまま再度「ZIPダウンロード」
- **方法2:** ZIPに含まれる `_FAILED_DOWNLOADS.txt` を **「失敗リスト読込」** ボタンで読み込むと、該当画像だけにチェックが入ります

## ダウンロードされるファイル

ZIPファイルの中身は以下の構成です：

```
avalab_images_20260316_0030.zip
├── avalab_20260221_wewZB1M9Q5ek3jB5Ir34pg.png    ← 画像ファイル
├── avalab_20260221_wewZB1M9Q5ek3jB5Ir34pg.txt    ← プロンプト情報
├── avalab_20260220_MrUVcXkySD2NOVywpasRRw.png
├── avalab_20260220_MrUVcXkySD2NOVywpasRRw.txt
└── ...
```

### ファイル名の規則

```
avalab_[生成日YYYYMMDD]_[画像ID].拡張子
```

### プロンプトファイル (.txt) の内容

```
[Positive Prompt]
1girl,solo,black hair,brown hair,looking at viewer,...

[Negative Prompt]
(ある場合のみ)

[Settings]
Fidelity: 0.6
LoRA ID: kQd-148-RZ61C8KZhBFn7Q
Generated: 2026-02-21T22:49:36.389464Z
Image ID: wewZB1M9Q5ek3jB5Ir34pg
```

## トラブルシューティング

### 「トークン未取得」が表示される

1. avalab.ai にログインしているか確認
2. avalab.ai のページを **F5でリロード**
3. 拡張機能のタブで **「再取得」ボタン** をクリック

### 「トークンが期限切れです」と表示される

Firebase認証トークンは約1時間で期限切れになります。

1. avalab.ai のページをリロード
2. 拡張機能のタブで「再取得」→「画像一覧を取得」をやり直す

### 一部の画像のダウンロードに失敗する

- 大量ダウンロード時はS3の署名付きURLが期限切れになる場合がありますが、50件ごとにURLを自動更新するため基本的に問題ありません
- 過去に削除した画像はプロンプト情報のみ保存され、画像ファイルはダウンロードできません
- 失敗した画像は `_FAILED_DOWNLOADS.txt` にエラー詳細が記録されます
- **「失敗リスト読込」** ボタンで失敗した画像だけを選択してリトライできます

### 拡張アイコンが表示されない

1. Chromeツールバー右端の **パズルアイコン** をクリック
2. 「Avalab.ai 一括ダウンロード」の **ピン留めアイコン** をクリック

## 技術仕様

| 項目 | 内容 |
|---|---|
| Manifest Version | 3 |
| API エンドポイント | `https://api.avalab.ai/v1/user/current/generated-image` |
| 認証方式 | Firebase Auth (Bearer Token) |
| 画像ストレージ | AWS S3 (署名付きURL) - 2つのバケットに対応 |
| ZIP生成 | JSZip ライブラリ |
| ページネーション | `l` (limit) / `o` (offset) パラメータ |

### ファイル構成

```
avalab-save/
├── manifest.json      # Chrome拡張マニフェスト
├── background.js      # Service Worker（トークン保持・API通信）
├── content.js         # Content Script（Firebase トークン読み取り）
├── inject.js          # MAIN World Script（fetchフックでトークンキャプチャ）
├── popup.html         # 操作画面 HTML
├── popup.css          # スタイル
├── popup.js           # 操作画面ロジック
├── lib/
│   └── jszip.min.js   # ZIP生成ライブラリ
└── icons/
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```
